V2 (wildlife documentary release)

At the outset, a producer may face either a strong-interest week or a weak-interest week, with probability 0.5 for each case.
The producer learns which week it is.
The distributor does not observe whether audience interest is strong or weak.

If it is a strong-interest week, the producer chooses between the ordinary release plan and the premium release plan.
If the producer chooses the ordinary release plan in the strong-interest week, the final reward is 4 for the producer and 4 for the distributor.
If the producer chooses the premium release plan in the strong-interest week, the distributor then chooses between release option X and release option Y.
If the distributor chooses release option X there, the final reward is 6 for the producer and 4 for the distributor.
If the distributor chooses release option Y there, the final reward is 0 for the producer and 0 for the distributor.

If it is a weak-interest week, the producer chooses between the ordinary release plan and the premium release plan.
If the producer chooses the ordinary release plan in the weak-interest week, the final reward is 4 for the producer and 2 for the distributor.
If the producer chooses the premium release plan in the weak-interest week, the distributor then chooses between release option X and release option Y.
The distributor cannot tell whether this decision follows a premium plan in the strong-interest week or in the weak-interest week.
If the distributor chooses release option X there, the final reward is 6 for the producer and 0 for the distributor.
If the distributor chooses release option Y there, the final reward is 0 for the producer and 6 for the distributor.
