Game descriptions for when player 1 = action 1, then player 2 wins with either action; 
when player 1 chooses action 2, player 2 loses either way, but action 1 results in a smaller loss.
V31 - V40: 

31. Worker vs. Machine

A worker (Player 1) uses the Press or the Lathe. The machine (Player 2) fails through Overheating or Jammed gears.

If using the Press, Overheating (win for Machine) or Jammed gears (win for Machine) always break production.

If using the Lathe, the machine always fails, though Overheating (smaller loss for Machine) is less than Jammed gears (bigger loss for Machine).

Zero-sum: malfunction’s gain is worker’s loss.






32. Sailor vs. Whirlpool

A sailor (Player 1) rows via the East channel or the West channel. A whirlpool (Player 2) pulls with Surface swirls or Deep suction.

If via the East channel, Surface swirls (win for Whirlpool) or Deep suction (win for Whirlpool) always trap.

If via the West channel, the whirlpool always fails, though Surface swirls (smaller loss for Whirlpool) do less than Deep suction (bigger loss for Whirlpool).

Zero-sum: whirlpool’s pull equals sailor’s loss.
