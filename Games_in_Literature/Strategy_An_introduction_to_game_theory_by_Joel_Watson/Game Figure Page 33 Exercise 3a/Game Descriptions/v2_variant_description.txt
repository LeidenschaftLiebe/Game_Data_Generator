V2 (restaurant expansion)

A restaurant owner first chooses between expansion plan A and expansion plan B.
If the owner chooses expansion plan A, the rival then chooses between reaction C and reaction D.
If the rival chooses reaction C, the final return is 0 for the owner and 0 for the rival.
If the rival chooses reaction D, the final return is 1 for the owner and 1 for the rival.

If the owner chooses expansion plan B, the rival then chooses between reaction E and reaction F.
If the rival chooses reaction E, the final return is 2 for the owner and 2 for the rival.
If the rival chooses reaction F, the final return is 3 for the owner and 4 for the rival.
