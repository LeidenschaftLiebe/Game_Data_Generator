Game Description 1: A Quiz Show Duel

In the fast-paced world of a quiz show duel, strategic choices are everything.

**Player1** must act quickly, choosing between:
- A calculated **buzzer quickfire**, or
- A daring **strategic pass**.

Facing them, **Player2** must respond with:
- A decisive **question steal**, or
- A clever **double points**.

The outcome depends entirely on how these strategies clash:

- If Player1 commits to **buzzer quickfire**:
  - And Player2 counters with **question steal**, the resulting payoff is (1, -1).
  - And Player2 counters with **double points**, the resulting payoff is (-1, 1).

- If Player1 commits to **strategic pass**:
  - And Player2 counters with **question steal**, the resulting payoff is (0, 0).
  - And Player2 counters with **double points**, the resulting payoff is (-2, 2).

Only the sharpest tactics will secure the win.
