Game Description 1: Billionaire Race To Space

In the heart of billionaire race to space, critical choices carve the path to triumph.

**Player1** evaluates the following strategies:
- rocket redesign
- branding stunt
- NFT orbit drop
- crew celeb leak
- launch countdown flip

Meanwhile, **Player2** plots their countermoves:
- wind cancel
- oxygen tank fail
- stream delay
- troll protest
- engine rumor

When both players commit to their choices, these outcomes unfold:

- If **Player1** selects **rocket redesign**:

  - And **Player2** counters with **oxygen tank fail**, the resulting payoff is (61, -61).
  - And **Player2** counters with **stream delay**, the resulting payoff is (46, -46).
  - And **Player2** counters with **troll protest**, the resulting payoff is (-76, 76).
  - And **Player2** counters with **engine rumor**, the resulting payoff is (64, -64).

  - And **Player2** counters with **oxygen tank fail**, the resulting payoff is (45, -45).
  - And **Player2** counters with **stream delay**, the resulting payoff is (-87, 87).
  - And **Player2** counters with **troll protest**, the resulting payoff is (-42, 42).
  - And **Player2** counters with **engine rumor**, the resulting payoff is (-52, 52).

- If **Player1** selects **NFT orbit drop**:
  - And **Player2** counters with **wind cancel**, the resulting payoff is (23, -23).
  - And **Player2** counters with **oxygen tank fail**, the resulting payoff is (65, -65).
  - And **Player2** counters with **stream delay**, the resulting payoff is (1, -1).
  - And **Player2** counters with **troll protest**, the resulting payoff is (73, -73).
  - And **Player2** counters with **engine rumor**, the resulting payoff is (91, -91).

- If **Player1** selects **crew celeb leak**:
  - And **Player2** counters with **wind cancel**, the resulting payoff is (-33, 33).
  - And **Player2** counters with **oxygen tank fail**, the resulting payoff is (-19, 19).

- If **Player1** selects **launch countdown flip**:
