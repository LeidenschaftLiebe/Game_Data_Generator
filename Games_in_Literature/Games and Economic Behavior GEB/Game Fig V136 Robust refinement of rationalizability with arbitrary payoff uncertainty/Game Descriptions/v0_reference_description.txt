This figure comes from the article’s example with “twins.”
The paper studies how an action can still be selected even when the set of possible later choices is not a singleton.
In the article’s example, the parameter is theta in {0, 2}, and the discussion then focuses on the complete-information type with theta = 0.

For this dataset instance, set theta = 0.

Player 1 moves first and chooses between U and D.

After that, Player 2 moves, but cannot tell whether Player 1 chose U or D.
At that point, Player 2 chooses between L, R, and X.

If Player 1 chose U and Player 2 chooses L, the result is 1, 3.
If Player 1 chose U and Player 2 chooses R, the result is 0, 2.
If Player 1 chose U and Player 2 chooses X, the result is 1, 1.

If Player 1 chose D and Player 2 chooses L, the result is 0, 2.
If Player 1 chose D and Player 2 chooses R, the result is 1, 3.

If Player 1 chose D and Player 2 chooses X, then Player 1 moves again.
At that last step, Player 1 chooses between u and d.

If the path is D, then X, then u, the result is 2, 1.
If the path is D, then X, then d, the result is 0, 1.
