1. Prisoner’s Dilemma

Two suspects must decide whether to stay silent or betray.

If both stay silent, each gets –1.

If one betrays while the other stays silent, the betrayer gets 0 and the silent one gets –10.

If both betray, each gets –5.






2. Stag Hunt

Two hunters can cooperate to hunt a stag or go alone for a hare.

If both hunt stag, each gets 4.

If one hunts stag while the other hunts hare, the stag hunter gets 0 while the hare hunter gets 3.

If both hunt hare, each gets 3.






3. Chicken (Hawk–Dove)

Two drivers face off, each deciding to swerve or stay straight.

If one swerves while the other stays straight, the straight driver gets 2 and the swerver gets 0.

If both swerve, each gets 1.

If both stay straight, both crash and receive –10.






4. Battle of the Sexes

Two partners must choose between going to the opera or to the football game.

If both attend the opera, the opera lover gains 2, while the football fan gains 1.

If both attend the football game, the football fan gains 2, while the opera lover gains 1.

If they split and go to different events, both receive 0.






5. Matching Pennies

Alice and Bob each choose heads or tails simultaneously.

If they choose the same side, Alice gains +1 while Bob receives –1.

If they choose different sides, Alice receives –1 while Bob gains +1.






6. Rock–Paper–Scissors (Alice vs. Bob)

Two players simultaneously choose rock, paper, or scissors.

If one plays rock and the other plays scissors, the rock player gains +1, while the scissors player receives –1.

If one plays scissors and the other plays paper, the scissors player gains +1, while the paper player receives –1.

If one plays paper and the other plays rock, the paper player gains +1, while the rock player receives –1.

If both choose the same action, both receive 0.
