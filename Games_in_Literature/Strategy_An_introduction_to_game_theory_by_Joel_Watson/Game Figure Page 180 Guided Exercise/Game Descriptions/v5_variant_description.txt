V5 (museum exhibition partnership)

A museum first chooses between exhibition plan A and exhibition plan B.

If the museum chooses exhibition plan A, a donor partner then chooses between response a and response b.
If the donor partner chooses response a, the final reward is 3 for the museum and 3 for the donor partner.
If the donor partner chooses response b, the final reward is also 3 for the museum and 3 for the donor partner.

If the museum chooses exhibition plan B, the donor partner then chooses between response c and response d.
If the donor partner chooses response c, the final reward is 0 for the museum and 4 for the donor partner.
If the donor partner chooses response d, the final reward is 0 for the museum and 3.5 for the donor partner.
