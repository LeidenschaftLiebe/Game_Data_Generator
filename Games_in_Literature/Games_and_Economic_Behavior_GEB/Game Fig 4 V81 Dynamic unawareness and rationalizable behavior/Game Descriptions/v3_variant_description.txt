V3 — Shipping Guidance

Nature first determines whether the real port condition is Open Channel or Blocked Channel.
After observing that condition, the port expert sends either Route 0 (0) or Route 1 (1).
The shipping manager then chooses between Option A (a0) and Option B (a1).

When the manager moves, the manager observes the expert’s route recommendation but does not observe whether the port
condition is Open Channel or Blocked Channel.
So after hearing Route 0 (0), the manager chooses between Option A (a0) and Option B (a1) without knowing
which condition occurred.
After hearing Route 1 (1), the manager again chooses without seeing the condition itself.

If the path is Open Channel, Route 0 (0), Option A (a0), the outcome gives reward 0 to the expert and reward 0 to the manager.
If the path is Open Channel, Route 0 (0), Option B (a1), the outcome gives reward -1 to the expert and reward -1 to the manager.

If the path is Open Channel, Route 1 (1), Option A (a0), the outcome gives reward 0 to the expert and reward -10 to the manager.
If the path is Open Channel, Route 1 (1), Option B (a1), the outcome gives reward -1 to the expert and reward -11 to the manager.

If the path is Blocked Channel, Route 0 (0), Option A (a0), the outcome gives reward -1 to the expert and reward -11 to the manager.
If the path is Blocked Channel, Route 0 (0), Option B (a1), the outcome gives reward 0 to the expert and reward -10 to the manager.

If the path is Blocked Channel, Route 1 (1), Option A (a0), the outcome gives reward -1 to the expert and reward -1 to the manager.
If the path is Blocked Channel, Route 1 (1), Option B (a1), the outcome gives reward 0 to the expert and reward 0 to the manager.
