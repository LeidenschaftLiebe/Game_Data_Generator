Game Description 1: Crime Scene Deduction

In the midst of crime scene deduction, strategies and instincts collide in a dramatic showdown.

**Player1** evaluates their possibilities:
- test fibers
- time lapse
- witness bluff

At the same time, **Player2** readies their countermoves:
- fake ID
- trace phone
- photo zoom

As choices are made, the outcomes reveal themselves:

- Should **Player1** choose **test fibers**:
  - If **Player2** responds with **fake ID**, the resulting payoff is (-14, 14).
  - If **Player2** responds with **trace phone**, the resulting payoff is (-13, 13).
  - If **Player2** responds with **photo zoom**, the resulting payoff is (-19, 19).

- Should **Player1** choose **time lapse**:
  - If **Player2** responds with **fake ID**, the resulting payoff is (-16, 16).
  - If **Player2** responds with **trace phone**, the resulting payoff is (-15, 15).
  - If **Player2** responds with **photo zoom**, the resulting payoff is (1, -1).

- Should **Player1** choose **witness bluff**:
  - If **Player2** responds with **fake ID**, the resulting payoff is (19, -19).
  - If **Player2** responds with **trace phone**, the resulting payoff is (5, -5).
  - If **Player2** responds with **photo zoom**, the resulting payoff is (-17, 17).

In this contest, adaptability and foresight shape the path to victory.
