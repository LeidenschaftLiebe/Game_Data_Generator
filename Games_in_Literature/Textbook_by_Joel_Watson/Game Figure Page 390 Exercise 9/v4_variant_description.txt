V4 (sports travel planning)

At the outset, a team may face a high-readiness situation with probability 2/3 or a low-readiness situation with probability 1/3.
The team manager learns which situation applies.
The travel coordinator does not observe whether readiness is high or low.

If the team is in the high-readiness situation, the manager chooses between plan X and plan Y.
If the manager chooses plan X in the high-readiness situation, the final reward is 3 for the manager and 0 for the coordinator.
If the manager chooses plan Y in the high-readiness situation, the coordinator then chooses between option L and option M.
If the coordinator chooses option L there, the final reward is 0 for the manager and 6 for the coordinator.
If the coordinator chooses option M there, the final reward is 6 for the manager and 0 for the coordinator.

If the team is in the low-readiness situation, the manager chooses between plan X and plan Y.
If the manager chooses plan X in the low-readiness situation, the final reward is 0 for the manager and 0 for the coordinator.
If the manager chooses plan Y in the low-readiness situation, the coordinator then chooses between option L and option M.
The coordinator cannot tell whether this choice follows plan Y in the high-readiness situation or in the low-readiness situation.
If the coordinator chooses option L there, the final reward is 3 for the manager and 0 for the coordinator.
If the coordinator chooses option M there, the final reward is 3 for the manager and 6 for the coordinator.
