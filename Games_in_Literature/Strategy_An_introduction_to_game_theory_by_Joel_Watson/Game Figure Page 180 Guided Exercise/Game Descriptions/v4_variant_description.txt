V4 (esports tournament deal)

An esports organization first chooses between tournament plan A and tournament plan B.

If the organization chooses tournament plan A, the event sponsor then chooses between move a and move b.
If the sponsor chooses move a, the final reward is 3 for the organization and 3 for the sponsor.
If the sponsor chooses move b, the final reward is also 3 for the organization and 3 for the sponsor.

If the organization chooses tournament plan B, the sponsor then chooses between move c and move d.
If the sponsor chooses move c, the final reward is 0 for the organization and 4 for the sponsor.
If the sponsor chooses move d, the final reward is 0 for the organization and 3.5 for the sponsor.
