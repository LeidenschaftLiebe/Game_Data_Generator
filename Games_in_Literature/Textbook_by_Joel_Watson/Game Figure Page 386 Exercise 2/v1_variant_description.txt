V1 (theme park VIP day)

At the outset, a park manager may face either a high-demand day or a low-demand day, with probability 0.5 for each case.
The manager learns which kind of day it is.
The operations chief does not observe whether demand is high or low.

If it is a high-demand day, the manager chooses between opening the standard plan and opening the premium plan.
If the manager opens the standard plan on a high-demand day, the final reward is 4 for the manager and 4 for the operations chief.
If the manager opens the premium plan on a high-demand day, the operations chief then chooses between staffing option X and staffing option Y.
If the operations chief chooses staffing option X there, the final reward is 6 for the manager and 4 for the operations chief.
If the operations chief chooses staffing option Y there, the final reward is 0 for the manager and 0 for the operations chief.

If it is a low-demand day, the manager chooses between opening the standard plan and opening the premium plan.
If the manager opens the standard plan on a low-demand day, the final reward is 4 for the manager and 2 for the operations chief.
If the manager opens the premium plan on a low-demand day, the operations chief then chooses between staffing option X and staffing option Y.
The operations chief cannot tell whether this decision follows a premium plan on a high-demand day or on a low-demand day.
If the operations chief chooses staffing option X there, the final reward is 6 for the manager and 0 for the operations chief.
If the operations chief chooses staffing option Y there, the final reward is 0 for the manager and 6 for the operations chief.
