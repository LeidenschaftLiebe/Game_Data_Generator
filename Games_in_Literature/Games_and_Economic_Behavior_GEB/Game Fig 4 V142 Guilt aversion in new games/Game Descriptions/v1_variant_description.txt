V1 [Backer rebate]

A project lead, a finance officer, and a backer are involved in a funding decision.
The project lead moves first and chooses between StayOut (Out) and UseBackerFunds (In).

If the project lead chooses StayOut (Out), the interaction ends immediately.
The result is 170 for the project lead, 100 for the finance officer, and 30 for the backer.

If the project lead chooses UseBackerFunds (In), 25 units are taken from the backer’s own resources and handed to the finance officer.
The finance officer then chooses between SmallRebate (Left) and FullRebate (Right).

If the finance officer chooses SmallRebate (Left), the finance officer keeps 20 and sends 5 to the backer.
That transferred amount is doubled, so the backer receives 10.
The result is 170 for the project lead, 120 for the finance officer, and 15 for the backer.

If the finance officer chooses FullRebate (Right), the finance officer sends the full 25 to the backer.
That amount is doubled, so the backer receives 50.
The result is 170 for the project lead, 100 for the finance officer, and 55 for the backer.
