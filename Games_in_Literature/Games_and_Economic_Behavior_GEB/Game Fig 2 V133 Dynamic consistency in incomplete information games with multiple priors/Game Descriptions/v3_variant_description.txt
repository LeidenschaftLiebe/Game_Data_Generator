V3 [Political coalition]

Two parties, A and B, are negotiating a coalition, and a larger bloc C reacts afterward.
The political environment is either state I or state II, each with probability 1/2.

Party A first chooses between BreakTalks (war) and SupportCoalition (peace).
A does not know the state.

If A chooses BreakTalks (war), then bloc C moves.

In state I, PunishA (p_A), StayNeutral (n), and PunishB (p_B) give 0, 10, 1; 6, 6, 0; and 10, 0, 0.

If A chooses SupportCoalition (peace), then party B chooses between RejectCoalition (war) and JoinCoalition (peace).

In state I, after RejectCoalition (war), PunishA (p_A), StayNeutral (n), and PunishB (p_B) give 0, 10, 0; 6, 6, 0; and 10, 0, 1.

In state I, after JoinCoalition (peace), FavorA (f_A) and FavorB (f_B) give 5, 4, 4 and 4, 5, 4.

In state II, after BreakTalks (war), PunishB (p_B), StayNeutral (n), and PunishA (p_A) give 10, 0, 0; 6, 6, 0; and 0, 10, 1.

In state II, after RejectCoalition (war), PunishB (p_B), StayNeutral (n), and PunishA (p_A) give 10, 0, 1; 6, 6, 0; and 0, 10, 0.

In state II, after JoinCoalition (peace), FavorB (f_B) and FavorA (f_A) give 4, 5, 4 and 5, 4, 4.
