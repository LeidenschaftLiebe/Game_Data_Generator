V2 (restaurant expansion)

A restaurant owner first chooses among expansion option A, expansion option B, and expansion option C.
The rival restaurant then chooses between reaction U and reaction D without knowing which option the owner selected.

If the owner chose option A and the rival chooses reaction U, the final return is 2 for the owner and 1 for the rival.
If the owner chose option A and the rival chooses reaction D, the final return is 1 for the owner and 2 for the rival.

If the owner chose option B and the rival chooses reaction U, the final return is 6 for the owner and 8 for the rival.
If the owner chose option B and the rival chooses reaction D, the final return is 4 for the owner and 3 for the rival.

If the owner chose option C and the rival chooses reaction U, the final return is 2 for the owner and 1 for the rival.
If the owner chose option C and the rival chooses reaction D, the final return is 8 for the owner and 7 for the rival.
