1. The Drivers’ Showdown

Two drivers barrel toward each other, deciding to swerve or stay straight.

If one swerves while the other stays, the straight driver gets 5, the swerver gets 0.

If both swerve, each gets 2.

If both stay straight, they crash and each suffers –20.

If both sidestep, each gains 1.

If both charge, they collide and each loses –15.
