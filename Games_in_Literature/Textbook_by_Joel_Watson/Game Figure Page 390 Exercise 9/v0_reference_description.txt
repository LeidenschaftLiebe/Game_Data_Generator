At the outset, the game is in case H with probability 2/3 and in case L with probability 1/3.
Player 1 learns which case has occurred.
Player 2 does not observe whether the game is in case H or case L.

If the game is in case H, Player 1 chooses between action X and action Y.
If Player 1 chooses action X in case H, the final reward is 3 for Player 1 and 0 for Player 2.
If Player 1 chooses action Y in case H, Player 2 then chooses between action L and action M.
If Player 2 chooses action L after Y in case H, the final reward is 0 for Player 1 and 6 for Player 2.
If Player 2 chooses action M after Y in case H, the final reward is 6 for Player 1 and 0 for Player 2.

If the game is in case L, Player 1 chooses between action X' and action Y'.
If Player 1 chooses action X' in case L, the final reward is 0 for Player 1 and 0 for Player 2.
If Player 1 chooses action Y' in case L, Player 2 then chooses between action L and action M.
Player 2 cannot tell whether the current decision point came after Y in case H or after Y' in case L.
If Player 2 chooses action L after Y' in case L, the final reward is 3 for Player 1 and 0 for Player 2.
If Player 2 chooses action M after Y' in case L, the final reward is 3 for Player 1 and 6 for Player 2.
