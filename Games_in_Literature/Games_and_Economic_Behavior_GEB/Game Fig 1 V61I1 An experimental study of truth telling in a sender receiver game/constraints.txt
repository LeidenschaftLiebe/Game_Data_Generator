Strategic Definition:
Better-Response Equivalence

Constraints:
1) Check the move order: chance moves first, then the Sender, then the Receiver.
2) Check the chance probabilities: the two states occur with probability 1/2 each.
3) Check that the Receiver has one information set after message A and one information set after message B, 
so the Receiver observes the message but not the true state.
4) Check the anchor payoff: the outcome after A -> A -> U is (2, 1).
