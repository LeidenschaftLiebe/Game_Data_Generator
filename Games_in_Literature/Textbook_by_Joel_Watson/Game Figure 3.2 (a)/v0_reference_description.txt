Player 1 moves first and chooses between action U and action D.
If Player 1 chooses action U, then Player 2 chooses between action A and action B.
If Player 2 chooses action A, then Player 3 chooses between action R and action T.
If Player 3 chooses action R, the outcome is 9 for Player 1, 2 for Player 2, and 5 for Player 3.
If Player 3 chooses action T, the outcome is 2 for Player 1, 4 for Player 2, and 4 for Player 3.
If Player 2 chooses action B, then Player 3 chooses between action P and action Q.

If Player 3 chooses action P there, the outcome is 0 for Player 1, 5 for Player 2, and 4 for Player 3.
If Player 3 chooses action Q there, the outcome is 3 for Player 1, 0 for Player 2, and 0 for Player 3.
If Player 1 chooses action D, then Player 2 chooses between action C and action E.
If Player 2 chooses action E, the game ends immediately with 6 for Player 1, 3 for Player 2, and 2 for Player 3.
If Player 2 chooses action C, then Player 3 chooses between action P and action Q.

At that point, Player 3 cannot tell whether the game reached this decision after action B or after action C.
If Player 3 chooses action P after action C, the outcome is 2 for Player 1, 2 for Player 2, and 2 for Player 3.
If Player 3 chooses action Q after action C, the outcome is 1 for Player 1, 2 for Player 2, and 2 for Player 3.
