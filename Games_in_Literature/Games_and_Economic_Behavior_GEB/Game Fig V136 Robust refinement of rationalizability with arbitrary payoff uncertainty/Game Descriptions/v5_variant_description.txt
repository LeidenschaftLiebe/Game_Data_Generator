V5 [Hiring process]

A manager moves first and chooses between FastTrack (U) and HoldBack (D).

Then the candidate responds, but the candidate cannot tell whether the manager fast-tracked or held back.
The candidate chooses between AcceptNow (L), WaitAndSee (R), and AskForRevision (X).

If the manager chose FastTrack (U) and the candidate chooses AcceptNow (L), the result is 1, 3.
If the manager chose FastTrack (U) and the candidate chooses WaitAndSee (R), the result is 0, 2.
If the manager chose FastTrack (U) and the candidate chooses AskForRevision (X), the result is 1, 1.

If the manager chose HoldBack (D) and the candidate chooses AcceptNow (L), the result is 0, 2.
If the manager chose HoldBack (D) and the candidate chooses WaitAndSee (R), the result is 1, 3.

If the manager chose HoldBack (D) and the candidate chooses AskForRevision (X), the manager moves again.
The manager then chooses between ApproveRevision (u) and RejectRevision (d).

HoldBack (D), AskForRevision (X), ApproveRevision (u) leads to 2, 1.
HoldBack (D), AskForRevision (X), RejectRevision (d) leads to 0, 1.
