Game Description 1: A Courtroom Cross-Examination

In the fast-paced world of a courtroom cross-examination, strategic choices are everything.

**Player1** must act quickly, choosing between:
- A calculated **timeline pressure**, or
- A daring **emotional witness**.

Facing them, **Player2** must respond with:
- A decisive **objection rise**, or
- A clever **discredit evidence**.

The outcome depends entirely on how these strategies clash:

- If Player1 commits to **timeline pressure**:
  - And Player2 counters with **objection rise**, the resulting payoff is (-6, 6).
  - And Player2 counters with **discredit evidence**, the resulting payoff is (5, -5).

- If Player1 commits to **emotional witness**:
  - And Player2 counters with **objection rise**, the resulting payoff is (6, -6).
  - And Player2 counters with **discredit evidence**, the resulting payoff is (7, -7).

Only the sharpest tactics will secure the win.
