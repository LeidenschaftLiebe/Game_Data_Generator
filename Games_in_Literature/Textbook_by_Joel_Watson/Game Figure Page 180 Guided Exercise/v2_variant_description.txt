V2 (book publishing deal)

An author first chooses between publishing route A and publishing route B.

If the author chooses publishing route A, the publisher then chooses between option a and option b.
If the publisher chooses option a, the final return is 3 for the author and 3 for the publisher.
If the publisher chooses option b, the final return is also 3 for the author and 3 for the publisher.

If the author chooses publishing route B, the publisher then chooses between option c and option d.
If the publisher chooses option c, the final return is 0 for the author and 4 for the publisher.
If the publisher chooses option d, the final return is 0 for the author and 3.5 for the publisher.
