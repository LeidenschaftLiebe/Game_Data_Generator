Game Description 1: Pirate Treasure Map Game

In the midst of pirate treasure map game, strategies and instincts collide in a dramatic showdown.

**Player1** evaluates their possibilities:
- dig east
- follow X
- mark tree

At the same time, **Player2** readies their countermoves:
- check compass
- read riddle
- split loot

As choices are made, the outcomes reveal themselves:

- Should **Player1** choose **dig east**:
  - If **Player2** responds with **check compass**, the resulting payoff is (-4, 4).
  - If **Player2** responds with **read riddle**, the resulting payoff is (-7, 7).
  - If **Player2** responds with **split loot**, the resulting payoff is (-12, 12).

- Should **Player1** choose **follow X**:
  - If **Player2** responds with **check compass**, the resulting payoff is (20, -20).
  - If **Player2** responds with **read riddle**, the resulting payoff is (3, -3).
  - If **Player2** responds with **split loot**, the resulting payoff is (12, -12).

- Should **Player1** choose **mark tree**:
  - If **Player2** responds with **check compass**, the resulting payoff is (13, -13).
  - If **Player2** responds with **read riddle**, the resulting payoff is (5, -5).
  - If **Player2** responds with **split loot**, the resulting payoff is (-15, 15).

In this contest, adaptability and foresight shape the path to victory.
