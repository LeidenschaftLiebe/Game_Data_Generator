Game Description 1: Robotics Contest Final

In the midst of robotics contest final, strategies and instincts collide in a dramatic showdown.

**Player1** evaluates their possibilities:
- turbo leg bot
- arm balance bot
- sensor scout

At the same time, **Player2** readies their countermoves:
- reaction AI
- heat sink
- terrain adapt

As choices are made, the outcomes reveal themselves:

- Should **Player1** choose **turbo leg bot**:
  - If **Player2** responds with **reaction AI**, the resulting payoff is (-8, 8).
  - If **Player2** responds with **heat sink**, the resulting payoff is (-19, 19).
  - If **Player2** responds with **terrain adapt**, the resulting payoff is (-6, 6).

- Should **Player1** choose **arm balance bot**:
  - If **Player2** responds with **reaction AI**, the resulting payoff is (9, -9).
  - If **Player2** responds with **heat sink**, the resulting payoff is (11, -11).
  - If **Player2** responds with **terrain adapt**, the resulting payoff is (12, -12).

- Should **Player1** choose **sensor scout**:
  - If **Player2** responds with **reaction AI**, the resulting payoff is (1, -1).
  - If **Player2** responds with **heat sink**, the resulting payoff is (19, -19).
  - If **Player2** responds with **terrain adapt**, the resulting payoff is (7, -7).

In this contest, adaptability and foresight shape the path to victory.
