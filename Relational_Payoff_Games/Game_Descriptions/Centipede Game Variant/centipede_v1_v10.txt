1) Startup Collaboration

Two founders alternate turns deciding whether to secure the current deal or pass to let the venture grow.
The deal ends when the decision comes back to the first founder.
If the first stops immediately, the first gets more and the second gets less.
If the first passes and the second stops, the second gets more and the first gets less.
If the second passes again and the first stops, the first now gets even more while the second gets a solid share.
If both pass until the end, they share the largest return equally.





2) Academic Collaboration

The decision ends when the turn cycles back to the first researcher.
Early stopping favors the stopper with a higher individual payout and a smaller one for the other.
If the first passes and the second stops, the second’s reward is higher than the first’s.
If the second passes again and the first stops, the first’s reward grows further.
If both pass to the end, they split the best outcome equally.
