V0 — Uncertainty-Averse Coordination Example

Player 1 moves first and chooses C or D.

If player 1 chooses C, then player 2 chooses c or d.

If player 2 chooses c, the interaction ends there.
Player 1 gets 1, player 2 gets 1, and player 3 gets 1.

If player 1 chooses D, the move passes to player 3.
If player 1 chooses C and player 2 chooses d, the move also passes to player 3.

When player 3 moves, player 3 cannot tell whether the move came from D or from C followed by d.
Player 3 then chooses L or R.

If player 3 chooses L, player 1 gets 3, player 2 gets 0, and player 3 gets 0.
If player 3 chooses R, player 1 gets 0, player 2 gets 3, and player 3 gets 0.
